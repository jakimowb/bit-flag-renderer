A QGIS Plugin to visualize bit flags in quality images.

Features

  • visualize single bit flags (on/off) and groups of neighbored bits (2 bits = 4 states, 3 bits = 8 states)
  • mapped bit flag states are listed in the layer legend
  • predefined flag schemes, e.g. for Landsat 4-8 quality bands
  • save and reload flag schemes

Resources

  • Documentation https://bit-flag-renderer.readthedocs.io/en/latest/
  • Repository https://bitbucket.org/jakimowb/bit-flag-renderer

The Bit Flag Renderer is developed at Humboldt-Universität zu Berlin, Earth Observation Lab.
